\section{Modular Exponentiation}

\begin{enumerate}
\item 
Compute the binary expansion of 2019\\
$2019 = 2(1009) + 1$\\
$1009 = 2(504) + 1$\\
$504 = 2(252) + 0$\\
$252 = 2(126) + 0$\\
$126 = 2(63) + 0$\\
$63 = 2(31) + 1$\\
$31 = 2(15) + 1$\\
$15 = 2(7) + 1$\\
$7 = 2(3) + 1$\\
$3 = 2(1) + 1$\\
$1 = 2(0) + 1$\\
\item
